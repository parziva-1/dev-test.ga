String slicing (Separando caracteres)

Posiblemente necesitemos obtener un caracter o una parte de los caracteres de un string.

Esta es la sintaxis general:

  • inicio es el indice del primer caracter que incluiremos en el slice. por defecto es 0.

  • final es el indice del ultimo caracter que vamos a incluir en el slice (este caracter no sera incluido). Por defecto, es el ultimo caracter del string (si omitimos este valor, el ultimo caracter sera incluido).

  • paso es cuanto vamos a sumar al indice actual para llegar al proximo.
